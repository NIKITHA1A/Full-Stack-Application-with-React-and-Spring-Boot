#  Full-Stack Todo Application (Spring Boot + React)

A full-stack Todo application built with:

-  Spring Boot (REST API)
-  React (Hooks + Axios)
-  PostgreSQL / H2
-  JWT Authentication (optional advanced setup)
-  Docker support

This project demonstrates clean architecture, REST conventions, validation, authentication, and modern frontend practices.

---

## Tech Stack

### Backend
- Spring Boot 3
- Spring Data JPA
- Spring Security (JWT)
- H2 / PostgreSQL
- Maven

### Frontend
- React 18
- Functional Components + Hooks
- Axios
- Context API (Auth)

### DevOps
- Docker
- Docker Compose

---

##  Features

### Backend
- Create Todo
- Get All Todos
- Filter by completed status
- Update Todo
- Delete Todo
- Title validation (mandatory)
- Layered architecture (Controller → Service → Repository)
- JWT Authentication (optional advanced setup)

### Frontend
- Display Todos
- Add new Todo
- Inline Edit
- Mark Complete / Incomplete
- Delete Todo
- Clean minimal UI
- Axios API integration
